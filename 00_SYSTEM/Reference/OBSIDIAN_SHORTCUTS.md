# Essential Obsidian Shortcuts

## Navigation

Cmd + O  
→ **Quick Switcher** (jump to any note instantly)

Cmd + Shift + F  
→ **Search the entire vault**

Cmd + [  
→ **Go back**

Cmd + ]  
→ **Go forward**

---

## Editing & Modes

Cmd + E  
→ **Toggle Reading / Editing**

Cmd + P  
→ **Command palette** (run any Obsidian command)

---

## Linking (MOST IMPORTANT)

[[  
→ **Create a note link**

Example:

[[Gliadin]]

Cmd + Click on a link  
→ **Open the linked note**

---

## Note Creation

Cmd + N  
→ **New note**

Cmd + Enter  
→ **Follow link / create note**

---

## Structure

# Heading 1

## Heading 2

### Heading 3

Example:

# Gluten  
## Components  
### Gliadins

---

## Graph & Knowledge

Cmd + G  
→ **Open Graph View**

---

# One rule to remember

Your vault works like a **knowledge network**.

Every important concept should eventually be:

one concept = one note

Example:

Gluten  
Gliadin  
tTG antibodies  
Villous atrophy  
Coeliac disease

This is exactly what you’re already building.

---

# The 3 things to memorise first (really)

If you remember **only these**, you’ll already move quickly:

Cmd + O  → jump to any note  
Cmd + E  → edit/read  
[[ ]]    → create links

Those three alone are **about 70% of Obsidian use**.
