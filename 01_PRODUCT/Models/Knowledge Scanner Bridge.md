
This page connects the biological ontology in allaboutG with the scanner system.

The scanner should not define biological rules.

Instead it must inherit them from this knowledge base.

## Principle

Biological truth informs product logic.

## Examples

Gluten proteins → scanner must detect wheat, barley, rye.

Coeliac disease immune mechanism → scanner default profile must assume strict gluten avoidance.

Wheat allergy → separate user safety profile.

Cross contamination pathways → scanner must include contextual preparation warnings.
